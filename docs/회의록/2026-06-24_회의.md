## 1. 오늘의 논의 내용

### Agenda 1. 구현 진행 현황 확인
- 출력 화면을 보면서 프로젝트 구현 진행 상황을 함께 점검했다.

### Agenda 2. 마커 표시 개수 조절
- 지도에 표시되는 마커가 너무 많아(258개) 화면이 복잡해지는 문제를 확인했다.
- 우선순위를 기준으로 상위 30개만 표시하도록 조정하기로 했다. 

### Agenda 3. 위치 기반 카페 검색
- 현재는 사용자 위치와 무관하게 결과가 표시되는 점을 확인했다.  
- 사용자 위치를 기준으로 근처 카페를 찾는 기능을 구현하기로 했다. 
- 사용자의 위치 데이터는 localStorage로 다루기로 했다.

### Agenda 4. 상세 페이지 구성
- 상세 페이지에는 카페의 자잘한 정보보다 핵심 정보가 더 필요하다는 의견이 나왔다. 
- 상세 페이지에서는 5개의 카테고리 기준 점수와 리뷰를 보여주기로 했으며 전화번호 등 세부 정보는 제외하기로 했다.

### Agenda 5. 로그인 방식
- 로그인은 아이디/비밀번호 방식으로 너무 복잡하지 않도록 구현하기로 했다.

### Agenda 6. 로그인 정보 저장
- 로그인 정보는 데이터베이스에 저장하기로 했다. 
- 위치 정보는 저장하지 않고 바로 사용 후 폐기하는 방식으로 정리했다. (localStorage)

### Agenda 7. 발표 영상 촬영 분담
- 발표 영상은 구현한 사람이 담당 부분을 설명하는 방식으로 촬영하기로 했다. 

### Agenda 8. 다음 회의 일정
- 다음 회의는 이번 주 금요일(06/26) 20:00로 확정했다.
